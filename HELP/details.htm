Пошаговое руководство

Далее будет описано пошаговое выполнение работы програмы на примере таблицы Event

работа с остальными таблицами идентична

Отображение:

При загрузке програмы клиент посылает запрос на сервер для получение сущетсвующих сущностей в таблице.

В результате чего компонент DataGrid заполняеться данными с БД

Добавление:

Для добавления сущности к таблице служит кнопка "Добавить"

В результате появится пустая строка для ввода пользовательских данных

После того как пользователь введет свои данные, нужно нажать кнопку "Сохранить"

Обработчик отправит запрос на добавление после чего данные сохранятся в БД и при слеующем заходе на данныю вкладку мы увидем только что добавленную нами новую сущность

Таким образом мы выполнили добавление в таблицу новой строки прямо в таблице Data Grid View

Редактирование:

Операция редактирования происходит подобным образом к добавлению

Что бы редактировать строку таблицы достаточно просто отредактировать желательные ячейки соответственной строки

После введения данных нужно отправить запрос на сервер для обновления данных. Для этого как и в случае с добавлением стоить нажать на кнопку "Сохранить"

Обработчик отправит запрос обновления после чего данные сохранятся в БД и при слеующем заходе на данныю вкладку мы увидем что наша строка была успешно изменена

Удаление:

Для удаления строки нужно сделать желаемую строку активной, путем нажатия на нее

Нажать кнопку "Удалить"

В результате чего выбранная нами строка уджалиться из таблицы в Data Grid View и из сервера БД
